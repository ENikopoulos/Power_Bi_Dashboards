## 📈 Data Jobs Dashboard (V1 Data Exploration)

![Data Jobs DG](/Images/Job_Dashboard_Home.PNG)

### 🔧 Key Power BI Skills Utilized

- **Dashboard Layout & Design:** Created a clean, intuitive and visually structured report layout  
- **Power Query (ETL & Data Shaping):** Performed data cleaning, transformation and preparation  
- **Data Modeling:** Established table relationships for efficient data analysis  
- **Measures & Aggregations:** Used implicit measures and standard aggregations for insights  

- **Data Visualization:**
  - **Core Charts:** Bar, column, line and area charts for trends and comparisons  
  - **Map Visualizations:** Displayed geospatial data insights  
  - **KPI Cards:** Highlighted key metrics  
  - **Tables:** Presented detailed, structured data  

- **Interactivity & Navigation:**
  - **Slicers:** Enabled dynamic data filtering  
  - **Buttons & Bookmarks:** Supported navigation and view management  
  - **Drill-Through:** Allowed deeper exploration of specific data points
